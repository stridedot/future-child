# BOOK_SCOPE.md

> 这本书不是什么都能装。
> 以后你一定会不断想到"这个也重要，那个也重要"。
> 这份文件就是用来拦住的。

---

## ✅ 在范围内

| 领域 | 说明 |
|------|------|
| 家庭教育决策 | 家长如何判断、选择、评估教育方式 |
| 儿童发展规律 | 不同年龄适合学什么、怎么学——有证据的结论 |
| AI 对技能的影响 | AI 改变了什么、没改变什么——基于数据 |
| 家长决策心理 | 为什么家长会焦虑、跟风、被营销——以及怎么避免 |
| 教育投入的有效性 | 哪些钱花得值、哪些是浪费——基于研究 |
| 普通家庭的可执行方案 | 月入几千到几万的家庭都做得到的事 |

---

## ❌ 不在范围内

| 领域 | 为什么不在 |
|------|------------|
| 学校制度改革 | 家长无法直接改变，不属于"家庭决策" |
| 高考政策分析 | 太大，且政策变化快，书出版时可能已过时 |
| 国际教育比较 | 我们的读者不送孩子出国，不需要芬兰和新加坡 |
| 特殊儿童教育 | 需要专业背景，超出本书能力 |
| 家庭教育法律问题 | 不属于"教育判断"范畴 |
| 教育产业发展趋势 | 不是写给投资者和政策制定者的 |
| 具体学科教学方法 | 比如"怎么教孩子数学"——这是工具书，不是这本书 |

---

## ⚠️ 灰色地带（需要讨论的）

| 话题 | 可能涉及 | 判断标准 |
|------|----------|----------|
| 青少年心理健康 | 如果讲"如何判断孩子是否有心理问题" | 只在和家庭教育决策直接相关时纳入 |
| 夫妻教育观念冲突 | 如果讲"夫妻意见不一致时如何决策" | 可以，但控制在半章以内 |
| 隔代教育 | 如果讲"如何和老人沟通教育方式" | 可以，但聚焦在"判断"而非"沟通技巧" |

---

## 范围测试

以后想到一个新话题，用这三问测试：

1. **这个问题是家长自己能做决定的吗？**（不是→排除）
2. **回答这个问题需要这本书独有的框架吗？**（不需要→排除，市面上已经有了）
3. **一个普通家庭家长读完，下周就能用上吗？**（不能→至少不是第一本书的内容）
