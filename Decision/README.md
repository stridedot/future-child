# 中国家庭教育决策知识库（China Family Education Decision Base）

> 这不是一本书的附属品。书是这个知识库的 V1.0 公开发布版。
> 以后每一年，新研究出现、AI 变化、访谈更多家庭、自己孩子长大——都可以继续更新。

---

## 组织原则

**按决策组织，不按学科组织。**

家长不会搜索"儿童发展理论"。家长搜索"五岁学英语吗？"

所以每个文件对应一个真实的家庭教育决策。

---

## 文件结构

每个 Decision 文件包含六个部分：

1. **Question** — 家长真正在问的问题（不是学术问题，是决策问题）
2. **Evidence** — 所有相关研究的元分析/综述/高质量来源
3. **MyView** — 当前观点 + 可信度评分 + 理由 + 边界条件 + 反例
4. **Counter View** — 反方观点 + 他们的依据 + 我赞同什么 + 我不同意什么
5. **Case** — 真实案例（成功/失败 + 原因分析）
6. **Version** — 版本记录（观点变化 + 变化原因）
