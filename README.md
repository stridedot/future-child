# future-child

> **未来家庭最大的竞争力，不是拥有更多教育资源。而是拥有更好的教育判断。**

一本帮助中国普通家庭建立教育判断力的书。不是告诉你"孩子应该学什么"——是帮你看清楚：你的每一个教育选择，到底是谁在做决定。

如果你在家长群里看到"大家都在报"会心跳加速，如果你被"最后 3 个名额"逼着付过款，如果你花了三万块之后说不清到底解决了什么问题——这本书是写给你的。

---

## 全书结构（v1.0，约 35,000 字）

| 部分 | 章节 | 核心 |
|------|------|------|
| **前言** | — | 教育市场卖的不是课程，是时间、未来、别人、价格、过去。最后，它拿走的是你的判断。 |
| **Part 1** | Ch1-3 | 为什么今天做父母越来越难？ |
| **Part 2** | Ch4-5 | 怎么判断？（五问 + 信息免疫力） |
| **Part 3** | Ch6-10 | 是谁拿走了你的教育判断？（五种认知漏洞） |
| **结语** | — | 把判断力传给下一代。 |

### Part 3 五种判断权外包机制

| 章 | 外包给 | 核心问题 |
|----|-------|---------|
| Ch6 | 时间 | 为什么你总相信"错过就晚了"？ |
| Ch7 | 未来 | 为什么你总相信自己知道未来需要什么？ |
| Ch8 | 别人 | 为什么别人的决定会变成你的焦虑？ |
| Ch9 | 价格 | 为什么你总觉得花更多钱更安全？ |
| Ch10 | 过去 | 为什么你把"坚持"和"正确"混为一谈？ |

## 反馈
